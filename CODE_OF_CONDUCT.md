# Code Of Conduct

## Our Standard

We want a practical, welcoming learning community. Be respectful, patient with beginners, precise when correcting technical claims, and generous with context. Critique ideas and code, not people.

Unacceptable behavior includes harassment, discrimination, threats, deliberate humiliation, sharing private information, spam, and sustained disruption.

## Enforcement

Report conduct concerns privately to the repository maintainers through GitHub. Maintainers may edit or remove content and temporarily or permanently restrict participation when behavior harms the community. Reports will be handled as confidentially as practical.

This code applies in repository spaces and when someone represents this project in public.
